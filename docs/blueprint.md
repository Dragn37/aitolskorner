# **App Name**: AI Tools Korner

## Core Features:

- Global Layout: Dark theme, responsive design with header and footer for consistent navigation.
- Homepage Sections: Hero section, trending tools, and latest articles for a quick overview.
- AI Tool Listing: Grid of AI tool cards with name, description, category, and trending badge.
- AI Tool Detail Page: Individual tool detail page template to display additional tool info.
- Article Listing: Page listing articles with previews and links.
- Article Detail Page: Individual article page template with clean typography for readability.
- SEO Optimization: Semantic HTML and meta descriptions for improved search engine visibility.

## Style Guidelines:

- Primary color: Deep violet (#4C1D95) to evoke intelligence and sophistication.
- Background color: Dark charcoal (#232323) to support the dark theme and focus on content.
- Accent color: Electric purple (#A21CAF) for interactive elements and highlights.
- Headline font: 'Space Grotesk' sans-serif for a techy, modern look. Body font: 'Inter' sans-serif for clean readability.
- Clean, minimal layout with clear content hierarchy and a focus on readability.
- Simple, consistent icons to represent tool categories and navigation elements.
- Subtle hover effects and transitions for a polished user experience.
